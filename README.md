# Currículo Online

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE.md)

## Descrição

Este projeto é um currículo online simples e intuitivo, desenvolvido em HTML. O currículo é organizado em seções e permite que o usuário adicione informações sobre suas habilidades, experiências profissionais e formação acadêmica. O currículo é responsivo e segue as melhores práticas de acessibilidade.

## Recursos

- Organização em seções
- Adição de informações sobre habilidades, experiências profissionais e formação acadêmica
- Responsividade para todas as plataformas
- Acessibilidade para usuários com deficiência

## Instalação

1. Faça o download do projeto em [https://github.com/your-username/curriculo-online](https://github.com/your-username/curriculo-online)
1.1. Neste caso acesse:  [https://igorfaggiano.github.io/static-http-access/](https://igorfaggiano.github.io/static-http-access/)
2. Abra o arquivo `index.html` em um navegador da web

## Uso

1. Preencha as informações do currículo nas seções correspondentes
